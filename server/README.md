# Cine Manage Verse Backend

This is the backend for Cine Manage Verse, built with Node.js and Express.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run in development mode (with auto-restart):
   ```bash
   npm run dev
   ```
3. Run in production mode:
   ```bash
   npm start
   ```
